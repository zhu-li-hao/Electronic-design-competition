【*】 程序简介 
-工程名称：使用串口和按键控制舵机
-实验平台: 野火STM32 F103 霸道开发板
-MDK版本：5.30
-ST HAL库版本：1.7.6

【 ！】功能简介：
使用串口和按键控制舵机


【 ！】实验操作：
将舵机VCC(位于舵机三线中间)接至5V,舵机GND接至开发板GND,信号线(一般为橙黄色)连接至开发板的PB8引脚.
下载本程序，复位开发板即可。
按动KEY1\KEY2键或通过串口可以控制舵机工作.

【 ！！】注意事项：
无
